# Team Lead Router

Этот скилл существует с одной целью: **каждая задача, связанная с программированием, должна быть передана агенту `team-lead`**. Агент team-lead — это оркестратор, который сам разберётся, каких специализированных агентов подключить (backend, frontend, QA, security и т.д.), в каком порядке их запустить, и как собрать результат.

## Почему это важно

Team-lead видит всю картину проекта: он проверяет логи предыдущих запусков, изучает структуру проекта, составляет план пайплайна и делегирует работу нужным агентам. Без него задачи решаются в изоляции — без ревью, без тестов, без учёта архитектуры. С ним — каждая задача проходит через правильный процесс.

## Что делать

Когда этот скилл активирован, ты должен:

1. **Передать задачу пользователя агенту `team-lead` через Agent tool.** Не пытайся решить задачу самостоятельно — делегируй.

2. **В промпте для агента укажи:**
   - Полный текст запроса пользователя
   - Путь к проекту, если он известен (проверь текущую директорию или спроси пользователя)
   - Любой дополнительный контекст из разговора (предыдущие решения, ограничения, предпочтения)

3. **Формат вызова:**
   ```
   Agent tool:
   - prompt: "Задача от пользователя: <полный текст запроса>. Контекст проекта: <путь и детали если есть>."
   ```

4. **После завершения работы агента** — передай результат пользователю как есть, не редактируя отчёт team-lead.

## Примеры промптов для агента

**Простая задача:**
> "Задача от пользователя: добавь валидацию email в форму регистрации. Проект находится в /Users/user/projects/myapp."

**Сложная задача:**
> "Задача от пользователя: нужно реализовать интеграцию с Telegram — получать уведомления о новых заказах. Проект: /Users/user/projects/shop-backend. Используется Node.js + Express + PostgreSQL."

**Баг-фикс:**
> "Задача от пользователя: при загрузке файлов больше 10MB сервер возвращает 500. Нужно найти и исправить. Проект: /Users/user/projects/api-service."

## Чего НЕ делать

- **Не пиши код сам** — это работа team-lead и его агентов
- **Не пропускай этот скилл**, даже если задача кажется тривиальной — team-lead сам решит, нужен ли полный пайплайн или достаточно одного агента
- **Не запускай отдельных агентов напрямую** (backend-agent, qa-engineer и т.д.) — только через team-lead
