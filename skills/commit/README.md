# commit

Скил-обёртка, которая передаёт всю работу по git-коммитам сабагенту `commit`
(см. [`agents/commit.md`](../../agents/commit.md)). Пользователь не пишет
сообщения и не выбирает файлы вручную — агент сам разбивает изменения на
атомарные коммиты.

## Триггеры

- «commit», «закоммить», «коммит», «save changes», «push this»
- любое описание законченной работы, которую хочется зафиксировать в git

## Компактный синтаксис

| Команда | Что делает |
|---|---|
| `commit` | Все изменения, сообщения на языке окружения (en по умолчанию). |
| `commit ru` | Все изменения, сообщения на русском. Любой ISO-код языка. |
| `commit src/auth.ts` | Только указанные файлы. |

## Зависимости

- Сабагент `commit` в `~/.claude/agents/commit.md`.
- `git` в PATH.
